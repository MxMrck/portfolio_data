# Business Intelligence (BI) Analyst - Portfolio

## Projet 1 - 📊 Looker to Tableau Dashboard Revamp - Decathlon Canada
  
## 🎯 Objectives
This project was a full revamp of Decathlon Canada's existing suite of Looker dashboards and BigQuery data pipelines. 
The goal was to modernize reporting practices, improve readability for business users, and reducing maintenance needs from the data science team.

## Solutions 

### On the Business side

1.  Introduce meaningful charts
2.  Refine KPIs and results indicators
3.  Rethink information hierarchy for better decision-making
4.  Design a better user experience with dashboards

### On the Tech side

1.  Replace a resource-intensive, ~1-hour Spark job with a lightweight solution directly connected to Databricks SQL Warehouse
2.  Consolidate dozens of intermediate tables into three core tables
3.  Simply the data model and reducing maintenance overhead
4.  Rebuilt the pipeline architecture with reliability as a priority

### 🧰 Stack
·  Looker 

·  Tableau

·  BigQuery 

·  Databricks
